# GearRentPro

## Project Description
GearRentPro is a JavaFX and MySQL based equipment rental management system developed for CMJD 112 coursework.

## Technologies
- Java (OpenJDK 23)
- JavaFX
- MySQL
- JDBC
- IntelliJ IDEA

## Database Setup
1. Open MySQL
2. Run the SQL file:
   sql/gearrentpro.sql
3. Database will be created with sample data.

## How to Run
1. Open project in IntelliJ
2. Add MySQL Connector/J to libraries
3. Configure JavaFX VM options
4. Run MainApp.java
